Photon Cloud Account

    The Fusion VR Shared Multiplayer Demo uses Photon Fusion and the Photon Cloud.
    A free account for the Photon Cloud is needed.

    The "Photon Fusion Hub" window should pop up when you open the demo in the Editor for the first time.
    This will guide you to create the needed Fusion AppId.

    In short:
    - Sign in or register on: https://dashboard.photonengine.com
    - Create an app of type Fusion
    - Copy and paste the Fusion AppId into the PhotonAppSettings scriptable object (or the Fusion Hub window)
	- Create an app of type Voice
    - Copy and paste the Voice AppId into the PhotonAppSettings scriptable object (Fusion / Realtime settings menu)
